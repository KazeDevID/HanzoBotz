## Instalasi

### Untuk termux

```
> apt update && apt upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> pkg install libwebp
> pkg install yarn
> git clone https://github.com/KazeDevID/HanzoBotz

> yarn
> npm start
```
